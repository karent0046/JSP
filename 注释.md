# JSP

## 一.注释

在 JSP 中支持两种注释的语法操作，一种是显示注释，这种注释是允许客户端看见的； 另一种是隐式注释，此种注释是客户端无法看见的

1、显式注释

在客户端可以看到的注释

继承HTML风格的注释    &lt;!-- HTML注释 --&gt;		<!-- 这是HTML注释 -->

2、隐式注释

在客户端不可以看到的注释

1、JSP自己的注释     <%-- 这是JSP注释 --%>

2、继承Java风格的注释		// 单行注释    /*多行注释*/

JSP 的三种注释方式：

1） // 注释，单行注释 /* 多行注释*/

2）&lt;!-- HTML注释 --&gt;

3）<%-- JSP 注释 --%>
